"How can I learn more about the MATLAB programming language?" / see "How can I modify, have greater control/autonomy" The Wavelet Toolbox was designed to enable new-comers to [first-time users], user-friendly - the focus is on interactivity. However, MATLAB is a fully-fledged , and with its integrated development environment (IDE), there's nothing to stop you moving on to using the Command Window or Editor to programmatically modify figures, and even experiment with running the functions [scripts] (contained in folder ...). If this interests you, we recommend getting started reading the MATLAB documentation, and we hope you enjoy getting to know MATLAB and our Wavelet Toolbox a little more [intimately]!

ERRORS: As a last ditch resort, most errors in the toolbox functionality can be relieved by restarting the GUI, that is closing the analysis window and running the module .m file again. We understand this is a nuisance, but we are still developing the toolbox. If you come across any errors and other undesirable behaviour, please let us know by sending us Feedback.
