Basic Features

Workspace

The analysis window contains an arrangement of panels that together constitute the workspace for that module. This collection of elements includes user interface controls such as buttons, and panels containing axes and dialog boxes. Each module offers unique functionality, and as such the content of the workspace varies depending on the analysis method. However, most of the elements are generic to each of the [five] modules, and are shown highlighted in Fig. 3:

Imported Data
The data that you load into the workspace is displayed graphically in the axes of the Imported Data panel. If imported in the correct orientation, time in seconds is displayed on the horizontal axis (x-axis) as the independent variable, and the dependent variable, e.g. amplitude, is displayed on the vertical axis (y-axis).
Graph Limits
To precisely specify the signal time span for analysis and the amplitude displayed in the Imported Data axes, the graph limits Xlim for the x-axis and Ylim for the y-axis can be respectively stated. The signal range of Xlim should be specified in seconds, and the range for Ylim in the associated unit of measurement. The default values of Xlim displayed after data import are the time in seconds of the first and last recorded samples. The default values of Ylim displayed after data import are the global minimum and maximum of the signal amplitude plus an additional padding for optimum clarity. The range for each axis is controlled by entering the lower of the boundary values, followed by a comma and the higher boundary value, e.g. Xlim: 3.15, 1200.00 and Ylim: -35.1, 42.4.
Note: A specification of the Ylim values will not affect the signal amplitude range subject to module analysis functions [/analysis], and only serves to control the vertical range of the graph view.
To update the display of the Imported Data axes view, press the Refresh button. In a corresponding manner, if the zoom or pan tools are used to modify the axes view range, the Xlim and Ylim numerical values may be updated accordingly by pressing the Refresh button.

An attempt to enter a data limit with greater precision than that with which it was measured will automatically be replaced with the rounded value at the precision of the measurement.

The Length dialog box (not editable) displays the number of sample points in the range specified by the Xlim input.
Select Data
The time series loaded into the workspace are labelled automatically upon import. The signal number is assigned logically as the row number in the case of row-wise orientated data, and the column number in the case of column-wise orientated data. If you are exploring multiple time series, a given signal of the data set can be individually displayed and analysed by selecting it from the Select Data list. {{Functionality in development - choice to analyse single signal vs. all, or a selection}}
Plots
The numerical results calculated by each module are represented graphically in the Plots panel axes. {{Information on different plot types: coloured surface plots for 3D, line graphs}}. To display the results of a different signal, select it in the Select Data list.
Graph Options
To emphasise a coordinate or interval on the y-axis (typically the Frequency (Hz) axis for most modules), a dashed line marker for a given y-coordinate may be plotted which spans the time range currently in the axes view. To specify a y-coordinate to plot a line marker, enter the y-value in the Intervals box and press Enter. You can raise multiple markers by inputting y-values in any order and separated with a single space, e.g. Intervals: 1.489 0.05 9.3. To remove a plotted y-value marker, remove it from the input box and press Enter.
Note: If you zoom in or pan across the graph axes, then enter an interval to plot, the line marker will be drawn only over the range of time that is viewable in the expanded or translated axes view. To extend the line across the entire plot, reset the axes to the original view, then click inside the Intervals dialog box and press enter again (without removing the desired Frequency (Hz) coordinate).
The Status bar presents prompts for your next action, and displays progress updates of any calculations you initialise.
Data Import
The Data Length dialog box (not editable) displays the total length of the imported time series in seconds.
Calculate Button
{The functionalities offered by each module include calculations initialised by pressing the Calculate --- button, where --- is replaced by the corresponding [measure].}
Advanced Options
The parameter selection presented in the workspace is extended by various advanced options, the nature of which depend on the module. As you learn more about the analysis techniques offered by the Wavelet Toolbox, subtleties in their functioning and the role of parameters involved in the modules will become clearer. The choice of functions, parameter types and values offered in the Advanced Options panel extend the possibilities of data types you can explore, and the dynamical properties open for investigation.
